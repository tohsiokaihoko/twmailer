421	Service not available	サービス利用不能。TCPコネクションの切断。
	【再現方法】
		メールサーバとの接続断

450	Mailbox unavailable	メールボックスが利用できないため、メール処理が実行不能。 (メールボックスのBusyなど、一時的なトラブル)
	【再現方法】
		メールボックスをBusyにする

451	Error in processing	要求された処理の実行中にエラーが発生したため、処理に失敗。 (サーバ側のエラー)
	【再現方法】
		apigwのエラー(bug)　本来発生しない

452	Insufficient system storage	ファイルシステムの容量不足のため、処理の実行不能。メモリ不足。
	【再現方法】
		メールサーバのディスク不足

500	Syntax error	コマンドの構文エラー。コマンドが認識されなかった。
	【再現方法】
		存在しないmethodを指定
	
		※結合試験で必要

501	Syntax error in parameters or arguments	コマンドのパラメータ、引数エラー。
	【再現方法】
		必須parameters不足、parameterの型不正
	
		※結合試験で必要

502	Command not implemented	指定のコマンドがサポートされていない。
	【再現方法】
		未実装メソッドを実行

503	Bad sequence of commands	コマンドの実行順序の誤り。
	【再現方法】
		今回は発生しないかな？

504	Command parameter not implemented	指定のコマンドのパラメータが未定義。サポートされていない。
	【再現方法】
		未実装パラメータを指定
			未実装の場合、そのパラメータは無視して処理実行でもいい

550	Mailbox unavailable	宛先のメールボックスが利用できない、存在しないという理由で処理不能。 User unknownエラー。
	【再現方法】
		存在しないアドレスへメール送信
			検知できる？
	
		※正常運用で発生する

551	User not local	宛先に指定されたユーザはローカルに存在しない。 <forward-path>にメールを送信せよ。
	【再現方法】
		ローカルドメインの存在しないアドレスへメール送信
			検知できる？不要？

552	Exceeded storage allocation	記憶領域の割当限度を超えたため、処理不能。
	【再現方法】
		IMAPアカウントのQuota上限超過

553	Mailbox name not allowed	宛先のメールボックス名が許可されていないため、処理不能。 (宛先間違いのほか、送信者のメールアドレスに問題がある場合もあり)
	【再現方法】
		不正なメールアドレスに送信
			504,550との違いは？

554	Transaction failed	処理失敗。
	【再現方法】
		送信処理失敗

600	Login error	ユーザー名またはパスワード不正。
	【再現方法】
		Loginで存在しないユーザ名を指定
		Loginで不正なパスワードを指定
	
		※正常運用で発生する
		※結合試験で必要

601	Authentication failed	不正な認証トークン。
	【再現方法】
		不正な認証トークンを指定
	
		※結合試験で必要

602	Authentication timeout	認証トークン期限切れ。
	【再現方法】
		期限切れトークンを指定
	
		※正常運用で発生する
		※結合試験で必要

603	Forbidden access	アクセス権なし。（認証はOKだが、実行権限がない場合。Admin権限が必要なメソッド実行時に発生）
	【再現方法】
		アクセス権制御しないので、発生しない

604	Account exist	指定したアカウントが存在する。（Admin.addAccount 実行時指定したメールアドレスが既に存在している場合に発生）
	【再現方法】
		Admin.addAccountで存在するアドレスを指定
	
		※結合試験で必要

605	Account does not exist	指定したアカウントが存在しない。
	【再現方法】
		存在しないユーザ名を指定
			ログイン以外でmailAddressを指定するメソッドで発生

606	Message does not exist	指定したメッセージが存在しない。
	【再現方法】
		存在しないUIDを指定してMessage.fetchを実行
	
		※結合試験で必要

607	Message read error	メッセージ読み込みエラー。(I/Oエラー)
	【再現方法】
